## Featured Projects

### Vention Platform Migration

A comprehensive 6+ month infrastructure migration project transitioning the core platform from Heroku (PaaS) to AWS native services, enabling improved scalability, cost efficiency, and operational control.

**My Role:** Senior Software Engineer (S5) @ Vention

**Key Contributions:**
- Led end-to-end migration from Heroku to AWS ECS (Elastic Container Service) for containerized application deployment
- Containerized all applications using Docker for consistent deployment across environments
- Designed and implemented CI/CD pipelines to automate build, test, and deployment processes
- Implemented SOPS (Secrets OPerationS) for secure key management and secrets encryption
- Migrated caching layer from Redis Cloud to AWS ElastiCache Serverless and ElastiCache Cluster for improved performance and cost optimization
- Migrated databases from Heroku Postgres to Amazon RDS for better control and scalability
- Implemented Infrastructure-as-Code using Terraform for consistent, version-controlled infrastructure
- Designed and implemented full E2E testing framework integrated into CI pipelines
- Authored RFCs and technical documentation for migration strategy and security implementations
- Managed staging and production environment transitions with zero downtime

**Impact:** Reduced infrastructure costs by ~50%, improved system reliability and scalability, and established modern DevOps practices with automated testing and deployment pipelines

**Technologies:** AWS ECS, Docker, Terraform, RDS (PostgreSQL), ElastiCache Serverless, ElastiCache Cluster, SOPS, CI/CD, GitHub Actions, Playwright, Ruby on Rails, Next.js
